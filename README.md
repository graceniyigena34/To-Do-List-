## To-Do List Web Application

A simple, interactive, and responsive To-Do List Web Application built using HTML5, Tailwind CSS, and JavaScript.
This project allows users to efficiently create, organize, and manage daily tasks while ensuring data persists across browser sessions using LocalStorage.
